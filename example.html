    Carousel Plugin

    twCarousel is a javascript carousel plugin that allow you to scroll some medias formated in an html list elements, pictures, text or whatever. His particularity is it's in pure javascript, concepted at the origins for replace the plugin Cycle, during the suppression of the dependancy library, the famous jQuery. But in order to be efficient, the plugin keep only necessary functions matching for aim.

    Compatibility.

    Usable and testable from ie7, firefox 23.0.1, Opéra 12.16, Safari 4.0.2 and Chrome 28.0.1500.95 m

    Using it.

    Simple example :

  <div class="example1">

  </div>

    Javascript :

    allnodeExample1 = twCarousel.getElementsByClassName(document.body,"example1");
twCarousel.carouselize({noeud:allnodeExample1});

    CSS :

    #example1{width:100px;height:45px;overflow:hidden;border:1px solid #555;}
.example1{width:100px;}

    Principes :

    • For HTML structure, a wrap div (#example1) contain a child list of elements (.example1). This child list can include any kind of html element you want.
    • In javascript, after the script is call (<script src="yourpath/twCarousel.js"></script>) the plugin est in the "twCarousel" object, having several methods, holding getElementsByClassName method, allowing you to get and stock the list of child element you want to scroll. You just have to pass the classname in parameter of "noeud" (french for node).
    • in CSS, you just have to put a fix value for width and height to the parent element, and it will be the "window" where elements will scroll. Each child node should be have the same width, or it will outweight the window.

    Options and parameters

    There is enough parameters for deal interactions with a meter, some thumbnails, arrows and some others.

    • noeud (Fr. for "node")- An object listing DOM node intended to scroll.
    • thumbnail Object listing DOM node intended to be use as thumbnails. An automatic class "activeSlide" is automaticaly affect to the focused element.
    • thumbHover Boolean value for setting an event mouseover on the thumbnails. thumbnail parameter should be previously set for this parameter work.
    • endRotation Time in milliseconds until when rotation stop. By default, animation never end.
    • compteur (fr. for counter in the sense meter)- String with the ID attribute where the current slide number will be display.
    • btNext String with the ID of the DOM node on user will engage action "next slide".
    • btPrev String with the ID of the DOM node on user will engage action "previous slide".
    • timer Time in milliseconds between two scroll actions. By default, set to 2000.
    • interactStopAll Boolean value for indicate if twCarousel should stop when user click on any of the user interfaces elements, including arrows or thumbnails.
    • AnimFrequency Time in milliseconds, wich is the frequency using for the elements scrolling. By default, set to 30.
    • AnimSplitdistance Numerical value indicating distance in pixel, each slide doing during the animation in the time interval eventually setted just previoulsy (AnimFrequency). By default, setted to 1 pixel. This value is automaticaly auto increment during the animation, with 1 pixel adding each interval time.

    Functions added

    For twCarousel work, some very useful functions similar to some jQuery ones has been added. So, you can use it automaticaly in your project too if you include the twCarousel script. 

    • twCarousel.getElementsByClassName = function(wrapNode, classname)
      • wrapNode : the parent DOM object from search will be crawl (can be useful for avoiding to crawl the full DOM document).
      • classname : string of the classname you search.
      • action : sent you the nodeDom object.
    • twCarousel.setCssAttribute = function(domNode, keyValue)
      • domNode : domNode object you want to edit CSS.
      • keyValue : a string containing the CSS couple key/value you want to use. For an example : "position:absolute;left:25px;"
      • action : Will attribute this CSS to the DomNode you targeted, avoiding to clone property if some is already define online.
    • twCarousel.addClass = function(nodeToCheck, cl)
      • nodeToCheck : domNode object you want to add a classname.
      • cl : the string with classname you want.
      • action : will add the classname on the nodeDom you targeted.
    • twCarousel.removeClass = function(nodeToCheck, cl)
      • nodeToCheck : target a nodeDom object you want to remove a class, and remove this class.
      • cl : string with the classname to remove.
      • action : Will remove the className from the node.
    • twCarousel.attachAnEvent = function(evName,NId,fnName)
      • evName : String contening DOM event name. Ex : "mouseover". Warning, the compatibility cross browser is not manage, except for the click event. It mean the difference "click"/"onclick" is manage, for others events, you have to be carreful or you can edit the plugin.
      • NId : the ID string you will associate to the event.
      • fnName : The function you will associate to the event.
      • action : Will allow you to associate a domNode and an event, for making any function you want.
    • twCarousel.detachAnEvent = function(evName,NId,fnName)
      • evName : A string with the event you want dissociate from a function you had previously associate with attachAnEvent.
      • NId : The domNode oject you want to dissociate.
      • fnName : The function name you want to dissociate from the event.
      • action : Will allow you do dissociate an event from a domNode.

    Examples

    With buttons for scrolling to next and previous.

    allnodeExample2 = twCarousel.getElementsByClassName(document.body,"example2");
twCarousel.carouselize({
	noeud:allnodeExample2,
	btNext:"arrowNext",
	btPrev:"arrowPrev"
});

    twCarousel with shorter show time for each slide.

    allnodeExample4 = twCarousel.getElementsByClassName(document.body,"example4");
twCarousel.carouselize({
	noeud:allnodeExample4,
	timer:1000
});

    With thumbnails (you can click it by default)

    allnodeExample3 = twCarousel.getElementsByClassName(document.body,"example3");
thumbnail3 = twCarousel.getElementsByClassName(document.body,"thumbs");
twCarousel.carouselize({
	noeud:allnodeExample3,
	thumbnail:thumbnail3
});

    With thumbnails (this time display in text) reacting to rollover

});

    After click on thumbnails or arrows, animation ending. 

    allnodeExample7 = twCarousel.getElementsByClassName(document.body,"example7");
thumbnail1 = twCarousel.getElementsByClassName(document.body,"thumbs1");
twCarousel.carouselize({
	noeud:allnodeExample7,
	thumbnail:thumbnail1,
	btNext:"arrowNext0",
	btPrev:"arrowPrev0",
	interactStopAll: true
});

    Exemple d'utilisation des paramètres AnimSplitdistance et AnimFrequency.

    allnodeExample8 = twCarousel.getElementsByClassName(document.body,"example8");
twCarousel.carouselize({
	noeud:allnodeExample8,
	AnimFrequency:100,
	AnimSplitdistance:2
});
